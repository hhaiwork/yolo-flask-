# yolo-flask-
使用yolov5+flask在网页上展示

1.从摄像头或者本地文件中获取视频流数据。

2.使用 YOLOv5 模型对每一帧视频进行目标检测，获取检测结果

3.将每一帧的检测结果绘制在对应的帧图像上，形成新的帧图像。
 
4.将新的帧图像序列编码为视频流传输到前端网页。 

5.在前端网页上播放接收到的视频流，并将每一帧解码后呈现到页面上。
